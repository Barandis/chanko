FixedBuffer

chanko/buffers~ FixedBuffer

Source:
  • buffers.js, line 161

A buffer implementation that never discards buffered items when a new item is added.

This buffer has a concept of full, but it's a soft limit. If the size of the buffer is exceeded, added items are still stored. full returns true any time the size limit is reached or exceeded, so it's entirely possible to call remove on a full buffer and have it still be full.
