SlidingBuffer

chanko/buffers~ SlidingBuffer

Source:
  • buffers.js, line 374

A buffer implementation that drops the oldest stored item when a new item is added to a full buffer.

This action is silent; the oldest item is simply removed before the new one is added. That means this buffer is never 'full' because it can always be added to without exceeding the size, even if that 'adding' also involves removing another item.
